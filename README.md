## 📖 简介 | Introduction

**OmniPrompt** 是一个汇集了高质量 System Prompt（系统提示词）的开源知识库（个人提示词汇总，README写这么高大上是因为是用ai写的）。旨在帮助开发者、内容创作者和 AI 爱好者更好地驾驭 Large Language Models (LLMs)。

本项目不仅仅是简单的提示词堆砌，而是涵盖了**编程开发、文档编写、创意设计**等多个垂直领域的结构化 Prompt 模板。每一条提示词都经过实战验证，确保能够引导 AI 输出高质量、符合预期的结果。

## ✨ 核心特性 | Key Features

*   **👨‍💻 编程开发深度优化**：包含架构设计、代码讲解、Bug 修复、爬虫编写等专业场景的指令。
*   **📝 文档自动化**：一键生成标准 README、技术文档润色、Markdown 排版优化。
*   **🎮 创意与游戏**：涵盖游戏设定（如 Minecraft）、项目起名灵感等趣味应用。
*   **🛠️ 元提示词能力**：内置 "提示词生成" 与 "提示词优化" 工具，帮你写出更好的 Prompt。
*   **📦 结构化管理**：按领域分类，易于查找和复用。

## 🛠️ 技术栈 | Tech Stack

*   **Core**: Markdown
*   **Target**: ChatGPT, Claude, Gemini, DeepSeek, etc.

## 🚀 快速开始 | Getting Started

### 1. 获取提示词
克隆本项目到本地：
```bash
git clone https://github.com/your-username/OmniPrompt.git
```

### 2. 选择场景
进入对应目录找到你需要的文件：

*   **编程相关**: `编程相关/Ai产品架构设计.md`, `编程相关/代码修bug.md`...
*   **Markdown工具**: `MarkDown相关/README生成.md`
*   **其他工具**: `其他/提示词生成.md`

### 3. 使用方法 | Usage Example

以 **"代码讲解"** 为例：

1.  打开 `编程相关/代码讲解.md` 文件。
2.  复制全文内容。
3.  在 ChatGPT/Claude 的对话框（或 System Prompt 设置区）中粘贴。
4.  **输入用户指令**：
    > "请讲解一下这段 Python 装饰器代码的原理..."
5.  AI 将严格按照设定的专家角色为您解答。
